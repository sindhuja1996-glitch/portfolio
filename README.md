# My Portfolio

Professional portfolio starter built with Next.js, React, and TypeScript.

## Run locally

1. Install dependencies with `npm install`
2. Start development with `npm run dev`
3. Open `http://localhost:3000`

## Customize

Update the placeholder profile values in `app/page.tsx`:

- `name`
- `email`
- `headline`
- `summary`

You can also replace the project cards with your real projects once you are ready.
